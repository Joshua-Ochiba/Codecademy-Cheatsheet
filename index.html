CSS SELECTORS TABLE REFERENCE

Table Tags

Syntax Name Description
selector { declaration } Type Selector The type selector matches the type of the element in the HTML document.
* { declarations } Universal Selector The universal selector selects all elements of any type.
.class name { declarations } Class Selector Selects elements of a specific class
class_name1{ declarations }
class_name2{ declarations }


Then, you could include both of these classes on one HTML element like this:


<element class= 'class_name1 class_name 2'> 		Multiple Classes This enables us to mix and match CSS classes.
#id { declarations; } ID Selector Uses the ID attribute of an HTML element to select one unique element on a page.
[attribute]{ declaractions } Attribute Seelector The attribute selector can be used to target HTML elements that already contain attributes.
selector:pseudo-class { property: value; } Pseudo-Class Selectors Used to add styles to selectors, but only when those selectors meet certain conditions.
